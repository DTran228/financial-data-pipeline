# Financial Data Pipeline

An end-to-end data engineering project that ingests financial market data, stores raw files, transforms datasets, and loads them into PostgreSQL for analytics.

## Tech Stack
- Python
- SQL
- AWS S3
- PostgreSQL
- GitHub

## Project Goals
- Build an ETL pipeline for financial market data
- Store raw and processed data
- Load cleaned data into a relational warehouse
- Run SQL analytics on returns, volatility, and trends
